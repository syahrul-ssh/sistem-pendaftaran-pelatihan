# Sistem Pedaftaran Pelatihan
## Cara Install setelah clone
1.  `composer install`
2.  `cp .env.example .env`
3.  `php artisan key:generate`
4.  `npm install`
5.  `npm run watch`
6.  `php artisan migrate:fresh --seed`
7.  `php artisan serve`
